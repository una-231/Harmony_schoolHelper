# Harmony_schoolHelper
- 本项目以**前端界面与交互逻辑实现**为主，采用模拟数据，不涉及后端接口。
校园助手前端页面
- 课程表查看  
- 校园公告浏览  
- 校园导览与地点推荐  
- 登录与个人中心管理
- 
## 🛠 开发环境

- 操作系统：Windows 10  
- 开发工具：DevEco Studio  
- 开发语言：ArkTS  
- UI 框架：HarmonyOS ArkUI  
- 应用模型：Stage 模型  
- 运行方式：HarmonyOS 预览器 / 模拟器
- ## 📱 功能模块说明

### 1️⃣ 登录模块（Login）
- 支持用户名密码登录（演示）
- 使用 `AppStorage` 管理登录状态
- 登录成功后进入首页，未登录状态自动拦截

### 2️⃣ 首页模块（Index）
- 展示应用核心功能入口
- 使用卡片形式呈现模块跳转
- 提供搜索按钮（演示）

### 3️⃣ 课程表模块（Timetable）
- 按星期展示每日课程安排
- 显示当前周次、日期与今日概览
- 支持空课程状态提示
- 课程卡片采用不同颜色区分课程类型

### 4️⃣ 校园公告模块（Notices）
- 公告按日期分组展示
- 支持“全部 / 未读 / 分类”筛选
- 支持关键词搜索
- 使用模拟公告数据

### 5️⃣ 校园导览模块（Guide）
- 展示校园主要地点信息
- 支持地点搜索与推荐
- 模拟地图展示与路线信息

### 6️⃣ 个人中心模块（Profile）
- 登录态校验
- 显示用户基本信息
- 提供设置入口（演示）
- 支持退出登录操作
